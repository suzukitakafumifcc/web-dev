## HTMLの概要

HTMLは、HyperText Markup Language（ハイパーテキストマークアップ言語）の略で、
ウェブページの構造を定義します。

要素によって構成されます。

### 要素

HTML要素の例を示します。

```html
<h1>見出し</h1>
```

上は`<h1>`要素（見出し）を表します。
見出しを囲む開始タグと終了タグのセットで要素となります。
